## Content

Form accepts one or more children to render as its form elements. Label alignment, positioning, and other [properties](#visual-options)
are set on the Form itself and propagated to its children to generate a cohesive design. To override these top level Form props on individual children,
set them directly on the child instead.

```tsx example
<Form maxWidth="size-3600" isRequired necessityIndicator="label">
  <TextField label="Name" placeholder="Jane Smith" />
  <TextField label="Email" placeholder="abc@adobe.com" />
  <TextField label="Address" placeholder="123 Any Street" isRequired={false} />
</Form>
```

## Labeling

### Accessibility

A label should be provided to the Form by adding either the `aria-label` or `aria-labelledby` prop, so that the element will be identified to assistive
technology as a form landmark region to which assistive technology can navigate.

```tsx example
<h3 id="label-3">Personal Information</h3>
<Form maxWidth="size-3600" aria-labelledby="label-3">
  <TextField label="First Name" placeholder="John" />
  <TextField label="Last Name" placeholder="Smith" />
  <RadioGroup label="Favorite pet">
    <Radio value="dogs">Dogs</Radio>
    <Radio value="cats">Cats</Radio>
    <Radio value="dragons">Dragons</Radio>
  </RadioGroup>
</Form>
```

## Events
Form accepts an `onSubmit` and `onReset` prop for handling form submission and resets respectively. In cases where your form elements' values are controlled, you must handle
resetting each element's value to their original default via the callback provided via `onReset`. This is particularly important when using form elements such as Radio, RadioGroup,
NumberField, and Switch whose values are inherently controlled through their implementation.

The example below demonstrates how one would go about handling Form submission and resetting.

```tsx example
import {Button} from '@react-spectrum/button';
import {ButtonGroup} from '@react-spectrum/buttongroup';

function Example() {
  let [name, setName] = React.useState('');
  let [email, setEmail] = React.useState('');
  let [role, setRole] = React.useState('Employee');

  let onSubmit = (e) => {
    e.preventDefault();
    alert('Submitting form!');
  };

  let onReset = (e) => {
    setName('');
    setEmail('');
    setRole('Employee');
  };

  return (
    <Form maxWidth="size-3600" aria-label="Sign up form example" onSubmit={onSubmit} onReset={onReset}>
      <TextField
        label="Full name"
        placeholder="John Doe"
        value={name}
        onChange={setName} />
      <TextField
        label="Email address"
        type="email"
        placeholder="123@abc.com"
        value={email}
        onChange={setEmail} />
      <RadioGroup
        label="Job"
        value={role}
        onChange={setRole}>
        <Radio value="Employee">
          Employee
        </Radio>
        <Radio value="Contractor">
          Contractor
        </Radio>
        <Radio value="Other">
          Other
        </Radio>
      </RadioGroup>
      <ButtonGroup>
        <Button variant="cta" type="submit">Submit</Button>
        <Button variant="secondary" type="reset">Reset</Button>
      </ButtonGroup>
    </Form>
  );
}

<Example />
```
